## System Overview

1. Image Acquisition
   - Industrial camera simulation
   - Controlled illumination assumed

2. Preprocessing
   - Noise reduction
   - Illumination normalization

3. Segmentation
   - Adaptive thresholding
   - Morphological filtering

4. Defect Detection
   - Connected component analysis
   - Area-based filtering

5. Metrology
   - Pixel-to-mm calibration
   - Dimensional measurements

6. Classification
   - Classical ML (SVM)
   - CNN-based proof-of-concept

7. Validation
   - Statistical analysis for repeatability
